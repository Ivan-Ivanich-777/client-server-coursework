# Клиент-Серверное приложение (Курсовой проект)

## Описание
TCP сервер и Qt клиент для решения математических задач с авторизацией пользователей.

## Структура проекта
-  + "Server/" + @ - Консольный TCP сервер на Qt
-  + "Client/" + @ - Графический клиент на Qt Widgets
-  + "Docs/" + @ - Документация, UML диаграммы, тесты
