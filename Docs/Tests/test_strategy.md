# Стратегия тестирования

## Модульное тестирование
- DBManager: регистрация, авторизация, статистика
- ServerCore: проверка задач (checkTask1/2/3)
- SocketManager: отправка/получение сообщений

## Интеграционное тестирование
- Клиент-сервер: все 4 типа запросов
- Авторизация с неверным паролем
- Регистрация существующего пользователя
- Проверка всех 3 задач с правильными/неправильными ответами

## UI тестирование
- Переключение между формами (Login → Reg → Dashboard)
- Ползунки X и A
- Зум и панорама графика
- Смена языка (EN/RU)
- Смена темы (Dark/Light)
- Изменение цветов
