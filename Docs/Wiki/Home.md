# TaMP Wiki — Главная

**TaMP** (Task and Math Platform) — учебный TCP-сервер с Qt-клиентом для авторизации, регистрации, проверки задач и ведения статистики.

## Краткое описание

- **Сервер** — консольное Qt-приложение, слушает порт 12345, обрабатывает текстовые команды через TCP.
- **Клиент** — Qt Widgets GUI с 4 формами: логин, регистрация, сброс пароля, задача.
- **База данных** — SQLite (таблицы `users`, `results`).
- **Протокол** — текстовый, UTF-8, разделитель `&`, терминатор `\n`.
