# Math Solver

Клиент-серверное приложение для построения графиков кусочно-заданной функции.
f(x) = { x^2 (x<0); x^3-3x (0<=x<2); x^4-4x^3+4x^2 (x>=2) }

## Особые точки
Минимум: (0,0) | Максимум: (-1/√3, 2/(3√3)) | Перегиб: (1,1)

## Технологии
Сервер: C++ Qt6 TCP:33333 | Клиент: Qt6 Widgets | БД: SQLite | Почта: Gmail SMTP
