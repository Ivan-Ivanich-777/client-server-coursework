Detailed Description

Виджет регистрации нового пользователя

Реализует трёхшаговый процесс регистрации:

  1. Ввод логина и пароля
  2. Ввод email
  3. Ввод OTP-кода подтверждения

Поддерживает валидацию полей, проверку доступности логина, блокировку при превышении лимита попыток ввода кода и навигацию между шагами клавишей Escape.

Создаёт все три шага регистрации с полями ввода и кнопками.

Создаёт трёхшаговую форму регистрации с карточкой, содержащей:
