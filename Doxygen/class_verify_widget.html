Detailed Description

Виджет двухфакторной верификации по OTP-коду

Экран ввода одноразового кода подтверждения, отправленного пользователю при входе в аккаунт. Поддерживает блокировку при превышении лимита попыток.

Обработчик нажатия кнопки «Подтвердить»

Хэширует введённый код и сравнивает с ожидаемым. При совпадении испускает сигнал verificationSuccess. При несовпадении уменьшает счётчик попыток и при достижении нуля блокирует ввод на 30 секунд.
