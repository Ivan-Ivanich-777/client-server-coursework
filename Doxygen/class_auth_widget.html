Detailed Description

Виджет авторизации пользователя

Предоставляет интерфейс для входа в аккаунт: ввод логина и пароля, отправка запроса на сервер, обработка ответа с блокировкой при превышении лимита попыток. Поддерживает переход к регистрации и восстановлению пароля.

void AuthWidget::onAuthResponseReceived ( const QString & response)

Слот для обработки ответа сервера на запрос авторизации

Parameters
responseОтвет сервера

Обрабатывает три варианта ответа:

  • auth_code_sent: сервер отправил код подтверждения, переход на верификацию
  • auth-: неверные учётные данные, увеличение счётчика попыток с блокировкой
  • другой: ошибка соединения с сервером
