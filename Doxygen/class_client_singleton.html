Detailed Description

Класс-одиночка для управления TCP-соединением с сервером

Обеспечивает единственный экземпляр клиента для подключения к серверу по протоколу TCP. Поддерживает синхронные и асинхронные запросы, а также уведомления о получении ответа через сигнал.

QString ClientSingleton::sendRequest ( const QString & request)

Отправить синхронный запрос и получить ответ

Отправляет строку запроса на сервер и блокирует выполнение, ожидая ответ. Если не подключён — пытается подключиться к 127.0.0.1:33333.

Parameters
requestСтрока запроса для отправки на сервер
Returns
Ответ сервера в виде строки или пустая строка при ошибке

Отправляет строку запроса на сервер с символом переноса строки в конце. Если клиент не подключён — пытается подключиться к 127.0.0.1:33333. Ожидает ответа сервера с таймаутом 60 секунд, после чего считывает все доступные данные.

Parameters
requestСтрока запроса для отправки на сервер
Returns
Ответ сервера в виде строки (без пробелов по краям) или пустая строка при ошибке подключения/таймауте

◆ sendRequestAsync()

void ClientSingleton::sendRequestAsync ( const QString & request)

Отправить асинхронный запрос

Отправляет строку запроса на сервер без блокировки. Ответ будет доставлен через сигнал responseReceived(). Если не подключён — пытается подключиться к 127.0.0.1:33333.

Parameters
requestСтрока запроса для отправки на сервер

Отправляет строку запроса на сервер без блокировки основного потока. Если клиент не подключён — пытается подключиться к 127.0.0.1:33333. При ошибке подключения испускает сигнал responseReceived с пустой строкой. Ответ от сервера будет обработан позже через слот onReadyRead.

Parameters
requestСтрока запроса для отправки на сервер
