Выполняет полную процедуру отправки email через SMTP.

Устанавливает SSL-соединение, выполняет авторизацию (AUTH LOGIN), формирует письмо с заголовками (Subject, From, To, Content-Type) и отправляет его. Завершает сессию командой QUIT.

Parameters
toEmailадрес получателя
subjectTextтема письма
bodyTextтекст письма
Returns
true, если письмо успешно отправлено; false — при ошибке

QString readEmailTxtValue ( const QString & key)
static

Считывает значение ключа из файла email.txt.

Ищет файл email.txt в каталоге исполняемого файла, на уровень выше и в текущем рабочем каталоге. Формат файла: ключ=значение Строки, начинающиеся с '#', игнорируются. Пробелы в значении удаляются.

Parameters
keyключ для поиска (например, "email" или "key")
Returns
значение ключа; пустая строка, если ключ не найден

Ожидает и считывает все доступные данные из SSL-сокета
