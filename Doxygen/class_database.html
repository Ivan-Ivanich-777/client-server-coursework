Detailed Description

Класс для работы с базой данных пользователей (Singleton).

Реализует паттерн Singleton для обеспечения единственного экземпляра подключения к SQLite базе данных users.db. Предоставляет методы для добавления пользователей, проверки учётных данных, управления электронной почтой и сбросом пароля.

Note
Конструктор, деструктор и оператор присваивания приватные — создание копий объекта запрещено.
