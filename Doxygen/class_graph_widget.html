Detailed Description

Виджет графического отображения кусочной функции

Позволяет визуализировать функцию f(x) с тремя ветвями на графике, управлять параметрами a, b, c через слайдеры и спинбоксы. Получает данные от сервера или вычисляет локально при недоступности сервера. Отображает таблицу значений и легенду.

Returns
значение функции f(x)

Три ветви:

  • x < -2: f(x) = |x*a| - 2
  • -2 <= x < 2: f(x) = b*(x^2) + x + 1
  • x >= 2: f(x) = |x - 2| + 1*c
